# Guide des captures d'écran à réaliser

Toutes les images doivent être placées dans le dossier `images/` (à côté de `main.tex`).
Format recommandé : **PNG**, largeur homogène (~1280px ou 1920px), fond clair.
Masquer toute donnée personnelle réelle — utiliser des données de démonstration.

## Recommandations techniques

- **Résolution** : 1920×1080 (ou 1280×800 au minimum)
- **Format** : PNG (pas JPG pour éviter la compression des textes)
- **Navigateur** : Google Chrome, zoom à 100%
- **Données** : utiliser une année de démonstration avec des données anonymisées ou fictives
- **Cohérence** : utiliser la même année universitaire sur toutes les captures
- **Qualité** : les textes des boutons et libellés doivent être lisibles
